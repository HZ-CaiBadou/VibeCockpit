# Changelog

All notable changes to VibeCockpit will be documented in this file.

## [3.2.8] - 2026-07-08

### Added
- Codex automatic wakeup settings and runtime scheduling.
- Three daily wakeup time slots, defaulting to 06:00, 11:00, and 16:00.
- Manual wakeup testing and wakeup history.

### Changed
- Rename the forked app to VibeCockpit.
- Use the bundle identifier `com.badou`.
- Focus the app on Codex usage monitoring and automatic wakeup.
- Replace the app icon and menu bar presentation.
- Show remaining Codex usage in the menu bar and detail popover.
- Remove Claude-facing UI from the main workflow for the current VibeCockpit build.
- Disable upstream VibeCockpit update feed usage for this fork.
